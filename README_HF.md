# Dynamic Causal Character Graphs

A lightweight structured-state framework for persistent fictional agents in long-form dialogue systems.

## Features

- **Belief Revision** — Log-odds belief updates from evidence
- **Causal Propagation** — Belief graph with A→B dependencies
- **Emotional State** — Valence/arousal tracking
- **Relationship Tracking** — Trust, affection, respect per entity
- **Smart Dialogue** — Contextually-aware character responses
- **No API Keys Required** — All features work out of the box!

## How to Use

1. Send messages to interact with the character
2. Watch beliefs, emotions, and relationships update in real-time
3. Use scenario presets for quick narrative events
4. Save/load your simulation state
